# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2024-06-07
### Added
- Initial Dispatch service with FastAPI application skeleton.
- Scrapers for Complex Shop, Universal Store, and GOAT with unified product schema.
- Security features: API key authentication and rate limiting.
- Telemetry client for runtime event tracking with optional external forwarding.
- Project documentation, configuration guidance, and packaging metadata.
